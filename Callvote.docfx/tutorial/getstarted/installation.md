# Installation
How to install ``Callvote`` on your server

> [!NOTE]
> This guide assumes you know **how to set up a SCP:SL server**.

### Choose your Plugin Loader
# [<span style="color:#43CCE5">LabAPI</span>](#tab/LabAPI)

### Download ``Callvote-LabAPI.dll`` from the Repository

[Click here to download.](https://github.com/Unbistrackted/Callvote/releases/latest/download/Callvote-LabAPI.dll)

<br><br>
### Install ``Callvote-LabAPI.dll`` into the <span style="color:#43CCE5">LabAPI</span> Folder

> [!WINDOWS]
> %AppData%\SCP Secret Laboratory\LabAPI\plugins\global

> [!LINUX]
> ~/.config/SCP Secret Laboratory/LabAPI/plugins/global

<br><br>
### Restart the server
While the server is loading, you should see the following message:

> [!]
> [20XX-XX-XX XX:XX:XX.XXX -XX:00] [INFO] [LabApi] [LOADER] Successfully enabled 'Callvote', Version: 6.6.3, Author: 'PatPeter & Unbistrackted'

This means that Callvote has loaded successfully!


# [<span style="color:#F53B3B">EXILED</span>](#tab/Exiled)

### Install <span style="color:#F53B3B">EXILED</span>

[Installation Process.](https://github.com/ExMod-Team/EXILED/blob/master/.github/documentation/README.md#installation)

<br><br>
### Download ``Callvote-EXILED.dll`` from the Repository

[Click here to download.](https://github.com/Unbistrackted/Callvote/releases/latest/download/Callvote-EXILED.dll)

<br><br>
### Install ``Callvote-EXILED.dll`` into the <span style="color:#F53B3B">EXILED</span> Folder

> [!WINDOWS]
> %AppData%\EXILED\Plugins

> [!LINUX]
> ~/.config/EXILED/Plugins

<br><br>
### Restart the server
While the server is loading, you should see the following message:

> [!]
> [20XX-XX-XX XX:XX:XX.XXX -XX:00] [INFO] [Exiled.API] Callvote v6.6.3 by PatPeter & Unbistrackted has been enabled!

This means that Callvote has loaded successfully!
